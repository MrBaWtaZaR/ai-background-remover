#### What it does
A simple web app that allows users to upload an image (PNG, JPG, WEBP) and get back a version of that image with the background removed, ready for download.

* **Front-end:** Streamlit
* **Back-end (AI):** `rembg` library
* **Image Processing:** `Pillow`
